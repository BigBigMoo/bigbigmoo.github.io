idea git提交项目，忽略文件或文件夹设置

2018年07月31日

idea提交的时候会出现一些不需要的文件，尤其是 .idea文件夹和.iml文件每次提交都要进行操作,很烦人
添加了gitignore貌似没作用，具体怎么设置过滤？
File —- Settings —- Editor – File Types
在 lgnore files and folders 中添加 .idea 以及 *.iml 然后点击 OK 就可以了!
(PS:注意添加时加上分号)
然后你就会发现 commit 的时候里面不会有那两个东西的
